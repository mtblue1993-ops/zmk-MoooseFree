# zmk-MoooseTB_Charlieplex
## 構成
* Charlieplex
* PMW3610
* Rotary Encoder
* zmk-rgbled-widget

## ピン配置
* &xiao_d 0 : Motion
* &xiao_d 1 : PIN1
* &xiao_d 2 : PIN2
* &xiao_d 3 : PIN3
* &xiao_d 4 : SDIO
* &xiao_d 5 : SCLK
* &xiao_d 6 : interrupt

* &xiao_d 10 : NCS
* &xiao_d 9 : PIN6
* &xiao_d 8 : PIN5
* &xiao_d 7 : PIN4

* &gpio0 9 : RE_B
* &gpio0 10 : RE_A
